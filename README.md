# Patreon (patreon)

Patreon is a membership platform for creators offering tiered subscriptions, content publishing, community, and payments. The Patreon API v2 follows the JSON:API specification and exposes campaigns, members (patrons), posts, tiers, benefits, and webhooks via OAuth 2.0.
